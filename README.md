# MiniRenderer

3D 软件光栅化渲染器



## 特性

- Qt 提供窗口绘制 和 相应事件

- 渲染管线（参考 OpenGL）
  - 右手坐标系，视图变换、透视&正交投影变换、视口变换
  - 线框模式：Bresenham直线算法
  - 填充模式：Scanline填充算法、重心坐标插值填充算法、透视校正插值
  - 顶点着色器、片段着色器
- 模型文件 obj 读取
- Blin-phong 光照模型、Normal Mapping
